Casos de Uso

Atendimento Humanizado com IA

Cliente envia mensagem no WhatsApp → Sistema aguarda 12 segundos para agrupar mensagens múltiplas → IA processa texto, transcreve áudios ou analisa imagens → Consulta memória de conversas anteriores e base de conhecimento (RAG) → Responde de forma natural, dividindo texto em mensagens curtas e gerando áudios quando apropriado → Marca como lida e registra interação.

Agendamento Inteligente Multi-Profissional

Cliente solicita agendamento → Sistema identifica o profissional desejado (Dr. Felipe ou Dra. Gabriela) → Consulta disponibilidade no calendário do GoHighLevel → Valida se há conflito de horários e aplica buffer de 30 minutos → Verifica se cliente já tem agendamento ativo → Cria novo agendamento ou oferece reagendamento → Atualiza pipeline do CRM e envia confirmação.

Reagendamento Sem Conflitos

Cliente solicita mudança de horário → Sistema localiza agendamento existente por email ou contact ID → Verifica disponibilidade do novo horário → Cancela agendamento antigo automaticamente → Cria novo agendamento com as mesmas informações → Atualiza registro no Supabase com novo Event ID → Move lead para etapa "Consulta Marcada" no pipeline.
