Fluxo Técnico: Bia SDR

Uma visão detalhada de como a automação processa leads, qualifica intenções e executa ações no CRM sem intervenção humana.

1. Entrada e Captura
O fluxo inicia quando um lead envia uma mensagem no WhatsApp. O Webhook recebe o payload, e o sistema imediatamente extrai o número de telefone e o conteúdo da mensagem. Se for um áudio, ele é transcrito para texto automaticamente.
Webhook OpenAI Whisper
2. Identificação e Contexto
O sistema consulta o GoHighLevel (GHL) para verificar se o contato já existe. Se existir, recupera o histórico da conversa e o estágio atual no pipeline. Se não, cria um novo contato. Isso garante que a IA tenha memória das interações passadas.
GHL API Supabase
3. Agente de IA (Cérebro)
O AI Agent analisa a intenção do usuário. Ele não apenas responde, mas decide qual ferramenta usar. Ele possui acesso a "tools" específicas:
  • Calculator: Para orçamentos.
  • Schedule: Para verificar disponibilidade de agenda.
  • Knowledge Base: Para tirar dúvidas sobre procedimentos (RAG).

OpenAI GPT-4o LangChain
4. Tratamento de Objeções
Se o cliente apresentar resistência (ex: "está caro", "vou ver com meu marido"), um módulo específico de Quebra de Objeções é ativado. A IA consulta uma base de melhores respostas persuasivas e tenta reverter a situação antes de desistir do lead.
Vector Store Prompt Engineering
5. Ação e Saída
Com a resposta gerada, o sistema executa a ação final: 1. Envia a resposta no WhatsApp (texto ou áudio gerado pela ElevenLabs). 2. Atualiza o status do lead no CRM (ex: move para "Interessado"). 3. Se houve agendamento, cria o evento no calendário.
WhatsApp API GHL Pipeline ElevenLabs
